隱私政策 服務條款

基本原則 本地優先

Cliplo 是一款剪貼簿管理工具，所有剪貼記錄均儲存於您的裝置本地資料庫，不會上傳至任何伺服器。

  • 無需帳號，無需登入
  • 不蒐集、不上傳任何個人資料
  • 剪貼內容、QR 掃描結果僅存於裝置本地，解除安裝後即一併刪除

無障礙服務（Accessibility Service）

Cliplo 的核心功能依賴 Android 無障礙服務來監聽剪貼簿變化，自動記錄您複製的文字內容。

  • 此服務僅用於偵測剪貼簿內容變化，不會讀取畫面上的其他內容（canRetrieveWindowContent 設為 false）
  • 偵測到的剪貼內容僅寫入裝置本地資料庫，不會上傳至遠端伺服器
  • 您可以隨時在 Android 設定 → 無障礙 中關閉此服務

相機使用（QR Code 掃描）

掃描 QR Code 功能需要存取相機畫面進行即時辨識。

  • 相機僅在您主動開啟掃描畫面時使用，離開掃描畫面即停止
  • 我們不錄製、不儲存、也不傳送任何相機影像
  • 掃描結果僅以文字形式存入裝置本地剪貼歷史

廣告（Google AdMob）

免費版本包含 Google AdMob 廣告，AdMob 可能蒐集以下資訊以提供個人化廣告：

  • 廣告識別碼（Android Advertising ID）
  • 裝置資訊（型號、作業系統版本）
  • 大略位置（根據 IP 位址）

詳情請參閱 Google 隱私政策。升級至 Pro 版可移除所有廣告。

服務說明

Cliplo 提供以下核心功能：

  • 剪貼簿管理：自動記錄複製的文字，依網址 / 電話 / Email / 程式碼 / 純文字自動分類
  • QR Code 掃描：相機掃描 QR Code，結果自動分類並存入剪貼歷史
  • 快速複用：點擊任一記錄即可重新複製到剪貼簿

功能限制說明

  • Android 10 以上系統限制非前景 App 讀取剪貼簿，因此部分情境需切回 Cliplo 才能補捉到剛複製的內容
  • 免費版上限為 50 筆記錄，超過上限會自動清除最舊的未釘選項目
  • 部分機型的省電模式可能影響背景監聽穩定性，建議將 Cliplo 加入電池最佳化白名單
